# Lark タスク依頼テンプレート集

このファイルは HONRAI_FACTORY でアルバイト 3 名に送信するタスク依頼のテンプレートです。  
各テンプレートは Lark チャットでコピペして使用できます。

---

## 🎨 Aさん向け：漫画セリフ確認

**用途**: 自動生成された漫画のセリフを確認・修正してもらう場合

```
【漫画セリフ確認依頼】

✨ 新しい漫画が生成されました。
確認・修正をお願いします。

📖 タイトル: [生成されたタイトル]
🎬 話数: 第 X 話
📝 ページ数: 12ページ

【確認項目】
✓ キャラクター名の一貫性
✓ セリフの自然さ・日本語の正確性
✓ ナレーション文の適切性
✓ シーン説明の正確性
✓ トーン・雰囲気の一致

📂 確認先: Google Drive / Notion
🔗 [リンク]

【期限】2026-05-27 18:00

【完了報告】
確認完了後、以下をコメントしてください:
✅ 「確認完了 - OK」
❌ 「修正が必要です - 詳細は [リンク] に記載」

質問があれば遠慮なくお聞きください！
```

## 使用例

### 方法 1: Lark チャットに直接コピペ

1. テンプレートをコピー
2. Lark グループチャットを開く
3. ペーストして、[名前]や[リンク]を置き換え
4. 投稿

### 方法 2: 自動トリガーから投稿

```python
# scripts/notify_tasks.py など
from lark.scripts.send_lark_notification import send_notification

task_template = """【漫画セリフ確認依頼】

✨ 新しい漫画が生成されました。
..."""

send_notification(task_template)
```

## テンプレートのカスタマイズ

各テンプレートは自由にカスタマイズできます。  
以下のプレースホルダーを置き換えてください：

| プレースホルダー | 置き換え内容 |
|-----------------|------------|
| `[名前]` | 人名やプロジェクト名 |
| `[ID]` | ファイルIDやタスクID |
| `[リンク]` | Google Drive / Notion のリンク |
| `[パス]` | ファイルシステムのパス |
| `[日時]` | 期限や発生時刻 |
