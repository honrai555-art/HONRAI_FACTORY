# Lark 通知テンプレート集

このファイルは HONRAI_FACTORY で使用される Lark 通知のテンプレートを集約しています。  
各テンプレートは `send_lark_notification.py` で送信できます。

**用途**: Google フォーム経由で新しい診断回答があった場合


## 使用例

すべてのテンプレートは以下の方法で Lark に送信できます：

### 方法 1: CLI から直接送信

```bash
python lark/scripts/send_lark_notification.py "✅ 新しい診断回答が入りました..."
```

### 方法 2: Python スクリプトから送信

```python
from lark.scripts.send_lark_notification import send_notification

# 通知テンプレートをそのまま使用
send_notification("""✅ 新しい診断回答が入りました

👤 回答者: 山田太郎
📅 回答日時: 2026-05-26 14:30
📝 診断名: キャリア適性診断""")
```

### 方法 3: 自動トリガーから送信

```python
# scripts/gpt_manga_generate.py など
if generation_completed:
    from lark.scripts.send_lark_notification import send_notification
    send_notification("🎨 漫画プロット生成完了...")
```
