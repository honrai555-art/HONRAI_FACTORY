# docs

`docs/` には HONRAI_FACTORY の設計、ワークフロー、運用手順をまとめます。

- プロジェクトドキュメント
- 生成フローの説明
- モデル・ツール構成
- 運用ガイドライン

## 主要ドキュメント

- `DISCORD_NOTIFICATION_ROLE.md` — Discordを遠隔操作の主役ではなく、完成通知・エラー通知・画像プレビューを受け取る通知センターとして定義します。
- `REMOTE_OPERATION.md` — ChatGPT / Codex / GitHubを中心にした遠隔運用と、Discord補助機能の位置づけを整理します。
- `SYSTEM_MAP.md` — HONRAI_FACTORY全体の役割分担を整理します。
- `WORKFLOW.md` — 製造フローとGitHub同期手順を整理します。
