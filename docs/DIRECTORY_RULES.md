# DIRECTORY RULES

## 目的

HONRAI_FACTORY のフォルダ構造を統一し、
AIと人間の両方が管理しやすい状態を維持する。

---

# 基本ルール

## 禁止

* final_final
* test2
* new_folder
* temp
* backup_old

など曖昧な命名は禁止。
