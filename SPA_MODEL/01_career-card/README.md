# 01_career-card — キャリアカード製造ライン

## 目的
キャラクター、職能、世界観タグをもとに、HONRAI世界の人物紹介・職業カード・SNS共有用カードを生成する製造ラインです。読者や制作チームがキャラクターの役割、成長軸、登場クエストをすばやく把握できる状態を作ります。

## 入力
- `SPA_MODEL/shared-data/character_rules.csv` のキャラクター命名・slugルール
- `SPA_MODEL/shared-data/tags.csv` のタイプ、属性、モチーフタグ
- `SPA_MODEL/shared-data/titles.csv` の肩書き・タイトル候補
- 追加の人物設定、職能、クエスト目的、出力形式の指示

## 処理
1. 共有データと対象キャラクターの既存カードを確認する
2. キャラクターの役割、スキル、ストーリー上の目的を整理する
3. カード本文、短い紹介文、検索用タグを生成する
4. 必要に応じてプレビュー画像やSNS向け文言を作成する
5. 成果物を所定の出力先へ保存し、不要な中間ファイルをGit管理対象にしない

## 出力
- カード原稿: `SPA_MODEL/01_career-card/output/` またはライン内で指定された成果物フォルダ
- 共有用データ: JSON / CSV / Markdown
- プレビュー: `SPA_MODEL/01_career-card/BuildPreviews/`（Git管理対象外）

## KPI
- 週次カード更新率
- 新規カード制作本数
- 既存カードの情報補完数
- キャラクターslugと共有データの整合率
- プレビュー確認完了数
