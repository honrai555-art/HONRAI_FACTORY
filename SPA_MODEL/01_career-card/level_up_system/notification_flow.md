# notification_flow.md

## 通知設計の目的

通知の目的は、ユーザーを成長DB入力へ誘導することではありません。

ユーザーが自分で設定したゴールに近づきたいと感じ、行動記録を行い、キャリアブロックを獲得することです。結果として、裏側の成長DBが増え、更新率が上がります。

## Ver3通知ゲームループ

```text
ゴール設定
↓
進路エラー通知
↓
ゴール連動クエスト通知
↓
行動記録
↓
キャリアブロック生成
↓
レベルアップ
↓
ゴール進捗上昇
↓
新しいクエスト
↓
行動記録
（循環）
```

## 1. ゴール設定通知

### 目的

ユーザーに現在ゴールを設定してもらい、ゲーム開始状態にする。

### 通知テンプレート

```text
【ゴール設定】
{user_name}さん、次に目指す「相応しいステージ」を設定しましょう。

例：
- 学習習慣を定着させる
- 志望校に合格する
- 月10万円の副業収益を作る
- 作品を販売する

ゴールを設定すると、次のクエストが自動生成されます。
```

## 2. 進路エラー通知

### 目的

ユーザーが設定したゴールと本来性のズレを伝え、次のクエストへ再接続する。

### 診断結果

- 現在地
- 本来性スコア
- ゴール一致率
- 進路エラー率
- 次のクエスト

### 通知テンプレート

```text
【進路エラー通知】
{user_name}さん、現在ゴール「{user_goal}」と本来性テーマ「{honrai_theme}」の間に、少しズレが出ています。

現在地：{current_position}
本来性スコア：{honrai_alignment_score}
ゴール一致率：{goal_match_rate}%
進路エラー率：{route_error_rate}%

次のクエスト：{next_quest}
今日の行動を1つ記録すると、キャリアブロックを獲得できます。
```

## 3. ゴール連動クエスト通知

### 目的

ユーザーゴールに応じたクエストを提示する。

### 通知テンプレート

```text
【ゴール連動クエスト】
現在ゴール：{user_goal}
クエスト名：{quest_title}
今日やる行動：{today_action}
獲得できるブロック：{target_block_type}
ゴール進捗への影響：{goal_progress_effect}
レベルアップ条件：{level_up_condition}
```

## 4. 行動記録通知

### 目的

行動記録を促し、保存時にキャリアブロックとゴール進捗を更新する。

### 通知テンプレート

```text
【行動記録通知】
{user_name}さん、今日の行動を記録して、ゴール「{user_goal}」に近づきましょう。

入力すること：
1. 今日やったこと
2. 気付き
3. 成果
4. ゴールへの進捗

保存すると、キャリアブロックとゴール進捗率が更新されます。
```

## 5. キャリアブロック獲得通知

### 目的

行動入力が保存された直後に、ブロック獲得の報酬感を出す。

### 通知テンプレート

```text
【キャリアブロック獲得】
{user_name}さん、新しいブロックを獲得しました。

対応ゴール：{user_goal}
種類：{block_type}ブロック
タイトル：{block_title}
ゴール進捗：{goal_progress_rate}%

このブロックは、Lv.{next_level} への条件に加算されます。
```

## 6. レベルアップ通知

### 目的

ユーザーゴールの進捗とキャリアブロックの積み上げにより、レベルと冠位の上昇を祝う。

### 通知テンプレート

```text
【レベルアップ通知】
{user_name}さん、レベルアップしました。

現在ゴール：{user_goal}
新レベル：Lv.{current_level}
現在冠位：{current_rank}
ゴール進捗率：{goal_progress_rate}%
理由：{level_up_reason}

次のクエスト：{next_quest}
```

## 学習塾向け通知表現

学習塾向けには、仕事化ではなく継続学習を前面に出します。

```text
【学習クエスト】
今日の目標は「{user_goal}」に向けた一歩です。
今日やること：{today_action}
保存すると、学習ブロックとゴール進捗が増えます。
```

## フリーランス向け通知表現

フリーランス向けには、仕事化と収益化を前面に出します。

```text
【仕事化クエスト】
現在ゴールは「{user_goal}」です。
今日やること：{today_action}
獲得できるブロック：{target_block_type}
この行動は、仕事化・収益化への実績になります。
```
