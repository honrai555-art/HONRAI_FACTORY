# プロンプト: Discord bot — コードレビュー

---

## 依頼内容

以下の Discord bot コードをレビューしてください。**自動修正はせず**、指摘と改善案のみお願いします。

**レビュー観点:**
- セキュリティ（トークン漏洩、権限過多）
- Discord API / レート制限
- エラーハンドリング
- 可読性（初心者が追えるか）
- 変更が必要な場合は **優先度（高/中/低）** を付ける

**制約:**
- 日本語で説明
- 勝手に大規模リファクタ案だけを出さない（段階的な改善を提案）
- 指摘ごとに「なぜ問題か」を一行で説明

**対象ファイル:**
<!-- 例: bot/src/index.js -->
