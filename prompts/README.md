# prompts — AI への依頼テンプレート

このフォルダには、Cursor や Claude に **コピー＆ペースト** して使えるプロンプト例を置きます。

## 使い方

1. 目的に合った `.md` ファイルを開く
2. `<!-- ここを書き換え -->` の部分を自分の状況に合わせて編集
3. チャットに貼り付けて送信

## ファイル一覧

| ファイル | 用途 |
|----------|------|
| `discord-bot-new-feature.md` | 新機能追加の依頼 |
| `discord-bot-bugfix.md` | 不具合修正の依頼（原因説明を強制） |
| `discord-bot-review.md` | コードレビュー依頼 |

## 共通ルール（すべてのプロンプトに含める）

- 日本語で回答すること
- 修正前に原因を説明すること
- 依頼範囲外の大規模変更をしないこと
- Windows + PowerShell 前提でコマンドを書くこと
